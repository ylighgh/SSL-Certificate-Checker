# SSL-Certificate-Checker

## 介绍

通过读取 `WEBSITES` 文件里的网址，返回网址的响应状态码，如果是传输协议是 `https`，同时返回证书相关信息

## 使用

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行

```bash
python checker.py
```
